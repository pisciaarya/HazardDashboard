Documentation

1. System Overview

The system is a mini-portal to visualize and analyze hazard vulnerability in settlements of Nepal featuring four hazard visualization maps (rivers, settlements, earthquakes, roads, wildfires) with real-time data integration from four external APIs. It includes seven spatial analysis tools and administrative boundary visualization.

Users interact through layer toggling controls, symbol-click popups with feature details, and dynamic parameter adjustments for buffer distances and hazard thresholds.

2. Architectural Implementation

Database Tier

  • PostgreSQL with PostGIS extension
  • Stores administrative boundaries, infrastructure networks, and hazard datasets
  • Executes spatial operations: buffering, intersection detection, distance calculations
  • Manages geospatial data in WGS84 and UTM Zone 45N coordinate systems

Backend Tier

  • Python Flask with 15+ RESTful API endpoints
  • Integrates real-time data from USGS, WAQI, OpenWeatherMap, and Open-Meteo APIs
  • Processes spatial analysis logic and converts results to GeoJSON
  • Manages dynamic query parameters and coordinate transformations

Frontend Tier

  • HTML, CSS, JavaScript with Leaflet.js
  • Interactive layer control interfaces with dynamic popups
  • Parameter input controls for spatial analysis tools
  • Real-time data refresh mechanisms
  • Created Web structure and Designing

3. Hazard Assessment Modules

Flood Exposure Analysis

  • User-defined buffers around river networks
  • Identifies settlements/local units within flood zones
  • Calculates percentage area affected for administrative units

Earthquake Impact Simulation

  • Magnitude-dependent risk zones with dynamic radii
  • Quantifies affected settlements and districts
  • Visualizes impact areas on interactive maps

Wildfire Hotspot Detection

  • Filters fire incidents by confidence level and date
  • Identifies settlements within configurable proximity buffers

Road Disruption Impact

  • Simulates infrastructure failure scenarios
  • Identifies settlements within impact radii

4. Spatial Analysis Tools

6. Key Spatial Operations

  • Metric buffering for flood and earthquake impact zones
  • Spatial intersections to identify affected settlements
  • Proximity analysis for fire and pollution vulnerability
  • Coordinate transformations between WGS84 and UTM Zone 45N
  • Geometric length and area calculations for infrastructure density
